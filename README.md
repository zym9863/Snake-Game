# 🐍 贪吃蛇游戏 (Snake Game)

[简体中文](README.md) | [English](README_EN.md)


这是一个使用HTML5 Canvas和JavaScript实现的经典贪吃蛇游戏，具有现代化的界面设计和流畅的游戏体验。

## ✨ 游戏特点

- 🎨 美观的游戏界面，包含网格背景和渐变色蛇身
- 👀 蛇头带有眼睛，会根据移动方向变化
- ✨ 食物有高光和脉动效果，更加生动
- 🎮 支持键盘方向键和WASD控制
- 🏆 记录并显示最高分数
- ⏯️ 支持游戏暂停和继续
- 📱 响应式设计，适应不同屏幕大小
- 🔄 游戏结束后可以立即重新开始

## 🎮 操作说明

### 键盘控制

- 方向键（↑ ↓ ← →）或WASD键控制蛇的移动方向
- 空格键：暂停/继续游戏

### 按钮控制

- 开始游戏/重新开始：开始新游戏
- 暂停/继续：暂停或继续当前游戏

## 📋 游戏规则

1. 控制蛇吃到食物后，蛇身会变长，得分增加10分
2. 撞到墙壁或自身会导致游戏结束
3. 游戏会记录最高分数，存储在本地浏览器中

## 🛠️ 技术实现

- **前端**：HTML5, CSS3, JavaScript
- **绘图**：HTML5 Canvas API
- **图标**：Font Awesome
- **字体**：Google Fonts (Poppins)
- **数据存储**：localStorage (保存最高分)
- **响应式设计**：适应不同屏幕大小

## 🚀 如何开始

1. 克隆或下载此仓库
2. 在浏览器中打开 `index.html` 文件
3. 点击“开始游戏”按钮开始游戏

```bash
# 如果你有Python，可以使用以下命令快速启动一个本地服务器
python -m http.server
# 然后在浏览器中访问 http://localhost:8000
```
