# 🐍 Snake Game

[简体中文](README.md) | [English](README_EN.md)

This is a classic Snake game implemented using HTML5 Canvas and JavaScript, featuring a modern interface design and smooth gameplay experience.

## ✨ Game Features

- 🎨 Beautiful game interface with grid background and gradient snake body
- 👀 Snake head with eyes that change based on movement direction
- ✨ Food with highlight and pulsing effects for a more vivid experience
- 🎮 Support for keyboard arrow keys and WASD controls
- 🏆 Record and display highest score
- ⏯️ Support for game pause and resume
- 📱 Responsive design, adapts to different screen sizes
- 🔄 Immediate restart option after game over

## 🎮 Controls

### Keyboard Controls

- Arrow keys (↑ ↓ ← →) or WASD keys to control the snake's movement direction
- Space bar: Pause/Resume game

### Button Controls

- Start Game/Restart: Start a new game
- Pause/Resume: Pause or resume the current game

## 📋 Game Rules

1. When the snake eats food, its body grows longer and the score increases by 10 points
2. Hitting the wall or the snake's own body results in game over
3. The game records the highest score, stored in the local browser

## 🛠️ Technical Implementation

- **Frontend**: HTML5, CSS3, JavaScript
- **Drawing**: HTML5 Canvas API
- **Icons**: Font Awesome
- **Fonts**: Google Fonts (Poppins)
- **Data Storage**: localStorage (for saving high score)
- **Responsive Design**: Adapts to different screen sizes

## 🚀 Getting Started

1. Clone or download this repository
2. Open the `index.html` file in your browser
3. Click the "Start Game" button to begin

```bash
# If you have Python, you can use the following command to quickly start a local server
python -m http.server
# Then visit http://localhost:8000 in your browser
```
